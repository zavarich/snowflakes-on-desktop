# Dependencies
Install `npm` from [Node](https://nodejs.org/)\
Open the console as administrator and enter:
```
cd path/to/your/app
npm i
npm list --save-dev
```
If list empty, use:
```
npm i electron --save-dev
npm i electron-builder --save-dev
``` 
# Commands
For test start use `npm start`\
For build app use `npm run build`
